📱 AppSheet Integration

📌 วิธีเชื่อม AppSheet:

  1. สร้าง Google Sheet ชื่อ "eNGINNo_Data"
  2. เชื่อม Apps Script URL ด้านบนก่อน
  3. เปิด AppSheet → New App → from Google Sheets
  4. เลือก Sheet "eNGINNo_Data"
  5. AppSheet จะสร้าง Mobile App อัตโนมัติ
